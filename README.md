# SportBot Documentation Site

Статический сайт-документация для Telegram бота [@SquatMasterBot](https://tg.pulse.is/SquatMasterBot?start=68272210e7d3bb3e50024c24|user_referrer_id=676150892).

## Локальный просмотр

Откройте `docs/index.html` в браузере.

## Публикация на GitHub Pages

1. Включите GitHub Pages в настройках репозитория:
   - Settings → Pages → Source: Deploy from branch
   - Branch: `main`, Folder: `/docs`

2. Сайт будет доступен по адресу:
   ```
   https://denkaydunov.github.io/sportbot-site/
   ```

## SEO оптимизация

✅ **Реализовано:**
- Meta теги (title, description, keywords)
- Open Graph для соцсетей
- Twitter Card
- Schema.org (SoftwareApplication)
- Canonical URL
- robots.txt
- sitemap.xml
- Semantic HTML5
- Mobile-first responsive дизайн
- Быстрая загрузка (один HTML файл)

## Структура

```
docs/
├── index.html      # Главная страница (полное руководство)
├── robots.txt      # Правила индексации для поисковиков
└── sitemap.xml     # Карта сайта для поисковиков
```

## Следующие шаги для продвижения

1. **Добавить свой домен** (опционально):
   - Купить домен (например, squatmasterbot.com)
   - Настроить DNS: CNAME → denkaydunov.github.io
   - Добавить файл `CNAME` в `/docs/`

2. **Google Search Console**:
   - Зарегистрировать сайт
   - Отправить sitemap.xml
   - Мониторить индексацию

3. **Yandex Webmaster**:
   - Зарегистрировать сайт
   - Отправить sitemap.xml

4. **Контент-маркетинг**:
   - Статьи на VC.ru, Habr
   - Посты в Telegram-каналах о фитнесе
   - Reddit r/fitness, r/bodyweightfitness
   - Видео на YouTube
